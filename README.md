# Excel Tutor

Ứng dụng desktop học Excel với giao diện thực hành giống Microsoft Excel.

## Công nghệ

- Vite + Vue 3
- Electron
- MySQL
- JavaScript

## Cài đặt

```bash
npm install
```

## Chạy development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Cấu hình MySQL

1. Tạo database bằng file `database/schema.sql`
2. Cập nhật thông tin kết nối trong `src/main/database.js`
